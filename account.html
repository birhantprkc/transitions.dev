Install with the CLI

Run these in your project. The first time, each opens your browser to confirm this device — then your Pro plan is verified and the files are pulled in.

npx transitions-pro skill

Installs the Pro transitions as an agent skill (CSS, React, TypeScript) into ~/.claude/skills/transitions-pro/.

npx transitions-pro add --pro

Copies every Pro transition recipe straight into your project's ./transitions/.
